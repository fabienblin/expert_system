# Expert System
Backward chaining inference engine - Scholar project
Written in Golang (go1.13 darwin/amd64)
# Compilation / Install

Use makefile : make / make run

# Usage

With a file :
./bin/expert_system example_input.txt

or dynamically :
./expert_system
